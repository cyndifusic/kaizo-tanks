Kaizo Tanks Installation Guide

IMPORTANT:

I can only safely say that this level pack will function on the North America (NTSC-U) release of the game. If you want to play the level pack using a disc from another region, be warned that it has not been designed for those versions or tested on those versions.

Additionally, if you plan to play using a 4:3 aspect ratio (not widescreen), you must set the "Custom Titles" option to DISABLED; otherwise, loading Tanks will crash your console. Gib and I are working towards fixing this issue, but for now, you're going to have to play on 16:9 to view custom level titles.

Finally, if for whatever reason Riivolution gives you the option to enable ocarina codes, disable them. Otherwise, the memory patches will not work correctly.

In order to play the Kaizo Tanks level pack, you will need the following tools:

  • a Wii or Wii U
  • a legitimate Wii Play disc
  • an SD card

Got all that? Great! Here are the steps you'll need to take to get Kaizo Tanks up and running:

  1. Install the Homebrew Channel on your Wii/Wii U. (If you already have the Homebrew Channel installed, you can skip this step.) If you are using a Wii, follow this guide, and if you're using a Wii U, follow this guide. Once it's installed, the Homebrew Channel will allow you to run custom software from an SD card on your Wii or Wii U.

  2. Install Riivolution on your Wii/Wii U. (If you already have Riivolution installed, you can skip this step.) Riivolution is a piece of homebrew software that allows you to apply patches to your Wii games. You can get it here.

  3. Determine which version of Wii Play you have. Yes, believe it or not, Nintendo released multiple versions of this game. In order to figure out which one you have, all you have to do is start up the game and look at the title screen. If the copyright date just says "2006", you are on version 1.00. If the copyright date says "2006-2007", you are on version 1.01.

  4. Download the corresponding ZIP file to your version.

    Version 1.00 Download
    Version 1.01 Download

  5. Extract the contents of the downloaded ZIP file to the root of your SD card. If you are asked if you want to merge folders or replace files, choose "Yes".

  6. Once the files finish copying, put the SD card into your Wii or Wii U, insert your Wii Play disc, start the Homebrew Channel, and then start Riivolution from there. If everything works correctly, after a few moments, it should come up with a menu with two options: "Level Pack" and "Custom Titles", which are both disabled by default. Enable both of them (unless you're playing in a 4:3 aspect ratio; see the note at the top of this page.) Then, click Launch to start up the mod.

  7. Start up Tanks and have fun! Keep in mind that you'll need to play as a guest Mii if you don't want to write to your existing Wii Play save data.
